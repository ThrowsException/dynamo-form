# Form API

A really simple experiment to save arbitrary posts to DynamoDB

## Getting Started

```bash
# pyenv local 3.7.0
python -m venv env
source env/bin/activate
(env) pip install -r requires.txt
(env) ./bootstrap.sh
(env) source ./dev-environment
(env) python app.py
```

visit http://localhost:8888/form.html for a test form.
